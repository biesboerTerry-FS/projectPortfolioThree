# Spotify Integration

## Project Overview
A full-stack application that allows users to search for music using the Spotify REST Web API. Users can search Spotify, , retrieve data by artist, album, or show.
 
## Prerequisites
* Node.js (v18.0.0 or higher)
* npm (v9.0.0 or higher)
* A Spotify Developer Account
* MongoDB Atlas Account (for Mongoose integration)

## Getting Started
1. Clone the repo: `git clone https://github.com/biesboerTerry-FS/projectPortfolioThree#`
2. Install dependencies: `npm install`
3. Create a `.env` file (see `.env.dist`) for variable reference
